# Changelog: 2026-07-09

## Marble Racing 2D — Game hoàn chỉnh

**Mô tả**: User yêu cầu tạo game marble racing 2D trong 1 file HTML self-contained, không thư viện, đầy đủ tính năng arcade racing.

### Quyết định kỹ thuật
- **Platform**: Pure HTML/CSS/JS, không dependencies
- **Physics**: Custom circle physics engine (không dùng thư viện)
- **Track**: Procedural generation với Catmull-Rom spline
- **Audio**: Web Audio API — generate âm thanh real-time
- **Save**: localStorage cho coins, skins, settings
- **AI**: Path-following với 3 cấp độ skill + random mistakes
